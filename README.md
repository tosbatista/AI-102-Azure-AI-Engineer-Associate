# AI-102 – Azure AI Engineer Associate

Public study repository with structured notes and hands-on labs for the **AI-102: Designing and Implementing a Microsoft Azure AI Solution** certification.

## 🎯 Objective
- Consolidate technical study by exam domain
- Practice with reproducible, public-safe labs
- Build a professional learning portfolio for Azure AI

⚠️ **Security note**  
This is a public repository. No subscription IDs, keys, endpoints, or sensitive data are stored here.

---

## 📚 Study Modules (Microsoft Learn aligned)

1. **Develop Generative AI Applications on Azure**
2. **Develop AI Agents on Azure**
3. **Develop Natural Language Solutions on Azure**
4. **Develop Computer Vision Solutions on Azure**
5. **Develop AI Information Extraction Solutions on Azure**

Each module contains:
- Technical summary
- Official Microsoft Learn links
- Suggested hands-on labs

---

## 🧪 Hands-on Labs

Labs are organized by capability and aligned to AI-102 scenarios:
- Generative AI (RAG)
- AI Agents with tools
- NLP (CLU / Q&A)
- Computer Vision (OCR)
- Information Extraction + Search

---

## 📌 How to use this repo
1. Pick a module under `/modules`
2. Read the summary and official links
3. Execute the related lab under `/labs`
4. Register learnings and mistakes under `/notes`
